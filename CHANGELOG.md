# Changelog

All notable changes to mkdocs-material-ekgf will be documented in this
file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-06

### Added

- Initial release of mkdocs-material-ekgf theme
- Custom three-row header layout with EKGF logo, site title, and OMG
  branding
- Enhanced footer matching ekgf.org design
- Process cards with hero background images
- Theme cards with 4-column responsive layout
- Objective badge components for visual hierarchy
- Integrated search box in navigation tabs
- Cross-subdomain theme cookie synchronization
- Dark/light mode support with automatic image switching
- Comprehensive SEO meta tags (Open Graph, Twitter Cards, JSON-LD)
- ChatGPT-style table styling
- Enhanced blockquote styling
- Responsive design optimizations for mobile, tablet, and desktop
- Complete CSS theming based on ekg-principles design
- JavaScript utilities for dark mode and search integration

### Design Elements

- EKGF color palette with light/dark mode variants
- Custom header chrome with backdrop filter effects
- OMG logo integration with separator lines
- Footer grid layout with social links
- Card hover effects with elevation shadows
- Smooth transitions and animations

### Documentation

- Comprehensive README with installation and usage
- Integration guide for existing EKGF sites
- Card layout examples and documentation
- Troubleshooting section

### Technical

- Python package structure with setup.py and pyproject.toml
- Support for Python 3.8+
- Compatible with MkDocs 1.5.0+ and Material for MkDocs 9.0.0+
- Package data includes all HTML templates, CSS, and JavaScript
- MIT License
